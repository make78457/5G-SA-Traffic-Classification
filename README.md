# 5G-SA-Traffic-Classification
Research code for BiLCNet: encrypted traffic classification using 5G SA physical channel records

---

## Core notebooks

- **`Data_preprocess.ipynb`** ⭐
  Preprocesses raw log files and generates training-ready data (feature matrices + labels).
- **`BiLCNet.ipynb`** ⭐ 
  Trains the BiLCNet classifier and runs evaluation, including **multi-scenario learning** and **LODO** (leave-one-domain-out) testing across unseen gain settings.
